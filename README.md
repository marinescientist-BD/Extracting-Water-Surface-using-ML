# Extracting-Water-Surface-using-ML
This single block code runs for extracting water body of selected shp file, but it follows organized linear flow and it starts from RGB layer, FCC, MNDWI, WRI, RF, Graphical visualization of MNDWI Distribution, Graphical Visualization of Band reflectance, Extracting meta data of satellite images, and all meta data of final classified images.
